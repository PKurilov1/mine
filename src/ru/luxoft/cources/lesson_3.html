Приложение «Калькулятор»

Цели упражнения

Ознакомиться с базовым синтаксисом языка Java. Научиться составлять простейшие программы в процедурном стиле на Java. Научиться планировать и осуществлять рефакторинг приложения.

​

Описание

Среда разработки IntelliJ IDEA предоставляет весь необходимый функционал для разработки, отладки и запуска простейших Java приложений.

Необходимо освоить этот функционал путём создания простейшего приложения консольного калькулятора. Реализуемые функции: сложение, вычитание, умножение, деление, факториал. Сценарий работы: пользователь вводит с консоли 2 числа и знак действия над ними (знак факториала принимаем за "!"), программа выводит результат действия над обоими числами. Действия повторяются до тех пор, пока пользователь не введет символ "q" (в этом случае программа завершает свою работу).

2Создание логики вычислений

11
После функции main создайте отдельные функции для арифметических операций
private static int plus(int numberOne, int numberTwo)
{
	return numberOne + numberTwo;
}

private static int minus(int numberOne, int numberTwo)
{
	return numberOne - numberTwo;
}

private static int multiply(int numberOne, int numberTwo)
{
	return numberOne * numberTwo;
}

private static int divide(int numberOne, int numberTwo)
{
	return numberOne / numberTwo;
}

12
После созданных выше функций создайте отдельную функцию для факториала
private static int factorial(int numberOne)
{
	int result = 1;

	for (int i = 1; i < numberOne; i++)
	{
		result *= i;
	}

	return result;
}

13

		break;

		break;
}
